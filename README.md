# Building-an-AI-Powered-Adaptive-Assessment-and-Learning-Platform-Using-Generative-AI
AI-powered adaptive assessment and learning platform built with FastAPI, Streamlit, SQLite, and Ollama Phi3. Features include AI-based MCQ generation, automatic evaluation, student analytics, PDF simplification, AI tutor assistant, and adaptive learning recommendations using Generative AI and NLP.



Designed and developed interactive Power BI dashboards for educational analytics, implementing KPI cards, DAX measures, weak-topic analysis, student performance tracking, pass/fail analytics, score distribution, and trend analysis.
Built end-to-end analytics pipelines by exporting data from SQLite using Python and integrating it into Power BI for business intelligence reporting and interactive dashboard visualization.
Implemented relational data modeling, slicers, filters, drill-down analytics, and dynamic visualizations to enable real-time performance monitoring and data-driven educational insights.
